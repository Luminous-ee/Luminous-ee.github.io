Gridea+GitHub Pages搭建

  1. 创建 GitHub 账号，登录。

    GitHub

  2. 创建一个公开仓库，名为xxx.github.io。（xxx为你的 GitHub 用户名）

    下方选择Public

  3. 进入账户设置: Settings->Developer settings->Personal access tokens->生成新的token

    Note做一些描述，勾选上 repo 的相关权限，即可生成一个 Token。如下图：

    注意：生成之后记得把 Token 复制到你的本地，因为一旦关闭网页将不能再看到它。


开始Gridea配置

  1. Gridea 进行远程设置

  2. 点击「检测远程链接」，进行配置检查。

  3. 点击 Gridea 中的同步按钮即可。

    初次部署等待几分钟，之后就可以去浏览器访问你的域名（即xxx.github.io）

  4. 你可以做一些自定义配置

    • 自定义菜单
    • 任意使用应用内默认主题或任意第三方主题，强大的主题自定义能力

开启评论功能

Gridea 支持 Gitalk 和 Disqus 两种评论系统，这里使用 Gitalk 评论。

Gitalk 是一个基于 GitHub Issue 和 Preact 开发的评论插件。

  1. 点此注册：Register a new OAuth application

  2. 将 Client ID 和 Client Secret 复制保存到本地

    请务必现在复制您的Client Secret，关闭网页后你再也看不到它了。

  3. 在Gridea中设置、保存、同步，评论功能开启。

现在，开启你的博客之旅吧
