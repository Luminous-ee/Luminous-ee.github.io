uTools-黑科技神器不能错过！

2021-08-25 工具

你的生产力工具集-uTools

官网地址：uTools

  • uTools 是一个极简、集成插件、跨平台的桌面工具。
  • 自由集成丰富插件，快速匹配「场景功能」，用完即走。
  • 当你熟悉它后，能够提高生产效率，让你更加专注地改变世界。

观看 uTools 的介绍视频

关键功能

  • 通过快捷键『Alt+Space』，快速唤起 utools 窗口。

    支持英文、英文驼峰、中文拼音、拼音首字母来快速搜索，你只要记得一个大概的名字、拼音、英文缩写，直接输入快速找到你想要的。

  • 选中内容，单击「鼠标」中键，弹出超级面板

    你只需要安装一些很小但非常实用的插件，超级面板会根据当前选择的内容（文本、截图、文件、文件夹）自动匹配已安装的插件供你选择。

  • 一切皆插件，丰富的插件市场任你选择

    • uTools 最大的特点就是拥有强大的插件系统。
    • 现在已有 300+ 插件，每个插件解决一个具体场景的问题，简洁美观、即用即走。
    • 根据自己的需求挑选安装，组合成自己最称手的工具集合，为各种日常操作提供便利。

了解更多内容，前往官方帮助文档

uTools相关插件推荐

在窗口输入插件，选择插件市场，进入并开始挑选你需要的插件。

  1. 聚合翻译

    这个插件整合有道翻译、腾讯翻译、搜狗翻译、谷歌翻译、必应翻译、百度翻译、Deepl翻译、彩云小译，你可以自由选择。

    关键字： translate/翻译

  2. 剪切板

    支持一次复制多个文本、图片、文件等，保存最近 200 条记录，随时可以查找并粘贴。

    关键字：clipboard

  3. Markdown笔记

    • 与传统富文本编辑方式结合，实时预览、存储
    • 代码块支持 164 种语言
    • 支持 TODO 任务列表
    • 导出 Markdown、PDF、图片文件

    强烈推荐，笔者的博客就是使用该插件写作

  4. JSON编辑器

    • 通过这个插件，URL Params、XML、YAML 粘贴自动转为 JSON 格式
    • 支持 JSON转XML、JSON压缩、JSON双引号转义
    • 在底部输入框使用 Javascript 语法快速对 json 数据进行过滤

    关键字：json

  5. 本地搜索

    插件是基于 Everything 的本地搜索，插件内置了绿色版 Everything

  6. 人生进度条

    「人生如戏, 白驹过隙」,很有趣的插件

  • uTools插件市场非常丰富，在此不能逐一列举，更多有趣强大的插件需要你自行探索，相信一定不会让你失望！

  • 插件市场大致分为五类：效率工具，搜索工具，图像工具，开发者工具，创意工具。

  • 可以说uTools插件系统基本能解决你所想到的所有问题，这么多插件有很多优秀的开发者在共同开发维护，如果你有相关技能，也可以加入开发者队伍！
